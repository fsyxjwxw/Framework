# SpringBoot一站式功能服务提供框架
此框架主要针对SpringBoot项目各类功能做出封装，整合各类插件，提供简便快速用法；

### 当前版本 v0.0.1

### 包含功能：

1. 整合Mybatis Plus单表查询；
2. 整合Swagger接口文档；
3. 整合Druid配置多数据源；
4. 封装全局异常捕获；
5. 封装同字段对象间转换方法；
